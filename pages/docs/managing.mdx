# Managing

## SSH into Devices

Once a device is added into a project, the CLI can be used to SSH into the device. The project and device ID need to be defined to pick the device to SSH into.

```shell
$ deviceplane ssh --project <PROJECT_ID_OR_NAME> --device <DEVICE_ID>
```
